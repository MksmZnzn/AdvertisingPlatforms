
Инструкция по запуску проекта "AdvertisingPlatforms"

Описание
--------
Проект представляет собой WebAPI на ASP.NET 8, предназначенный для загрузки и поиска рекламных площадок по локации. 
Данные хранятся в оперативной памяти, загружаются из текстового файла.

Требования
----------
- .NET 8 SDK
- IDE: Visual Studio 2022 или JetBrains Rider (или другая с поддержкой .NET 8)
- Операционная система: Windows/Linux/macOS

1. Скачай проект

2. Открой решение в IDE
Открой файл AdvertisingPlatforms.sln через Visual Studio или другую IDE с поддержкой .NET 8.

3. Выстави проект запуска
В свойствах решения установи AP.WebApi(WebApplication) как проект запуска.

4. Построй решение
Собери проект:
- В Visual Studio: Build > Build Solution (Ctrl+Shift+B)
- В Rider: Build > Build Solution

5. Запусти API
Нажми F5 или кнопку запуска. Откроется Swagger-интерфейс по адресу:
https://localhost:5001/swagger

6. Загрузка данных
Используй POST /load метод в Swagger, чтобы загрузить данные из текстового файла 
(формат: Название:/локация1,/локация2,...).

Пример содержимого файла:
Яндекс.Директ:/ru
Ревдинский рабочий:/ru/svrd/revda,/ru/svrd/pervik
Газета уральских москвичей:/ru/msk,/ru/permobl,/ru/chelobl
Крутая реклама:/ru/svrd
Глобальная реклама:/

7. Поиск рекламных площадок
Вызови GET /search?location=/ru/svrd/revda для поиска подходящих площадок.
